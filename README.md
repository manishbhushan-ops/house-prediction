# 🏠 AI House Price Predictor

A Machine Learning web application that predicts house prices based on various features such as area, number of bedrooms, bathrooms, and more.

---

## 🚀 Live Demo

👉 https://ai-house-price-predictor-app.streamlit.app/

---

## 📌 Project Overview

This project uses **Machine Learning (Linear Regression)** to estimate house prices.
It demonstrates a complete ML pipeline including data preprocessing, model training, evaluation, and deployment using Streamlit.

---

## 🧠 Features

* 📊 Predict house prices instantly
* 🖥️ Interactive and user-friendly UI
* ⚡ Fast predictions with optimized model
* 📈 Model evaluation (MAE, R² Score)
* 📌 Feature importance analysis
* 🎨 Styled UI with custom CSS

---

## 🛠️ Tech Stack

* **Python**
* **Pandas** – Data processing
* **Scikit-learn** – Machine Learning model
* **Matplotlib** – Visualization
* **Streamlit** – Web app deployment

---

## 📊 Model Details

* **Algorithm:** Linear Regression

* **Features Used:**

  * sqft_living15
  * bedrooms
  * bathrooms
  * floors
  * sqft_lot15
  * sqft_living
  * sqft_lot

* **Evaluation Metrics:**

  * Mean Absolute Error (MAE)
  * R² Score

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/house-prediction.git
cd house-prediction
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the app

```bash
streamlit run app.py
```
